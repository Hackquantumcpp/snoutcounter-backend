### Directory Map

- `banned_pollsters.R` - Banned pollsters list.

- `generic_ballot.R` - Generic ballot average model.

- `governor_primary_ca.R` - **WIP**. Model for 2026 California governor primary.

- `historical_president_approval.R` - **WIP**. Model for historical presidential approval.

- `president_approval.R` - Model for presidential approval average over the course of Donald Trump's second term.

- `president_approval_RV.R` - Same as above, but for registered voter and likely voter samples.

- `president_issue_approval.R` - Same as above, but for presidential approval on the issues.

- `run_snoutcounter.R` - Script for running all SnoutCounter models.
